아프리카TV
VR라이브 서비스는
다양한 조작 기기를
지원합니다.

크롬 및 파이어폭스 브라우저에 최적화
되어 있습니다.

  • 마우스 좌측 버튼을 클릭 후 드래그 하여 영상을 제어 할 수 있습니다.
  • 키보드 스페이스바를 이용하여 영상을 재생/멈춤 할 수 있습니다. 방향키를 사용하여 영상을 제어할 수
    있습니다. 숫자키 입력시 영상의 중앙으로 이동됩니다
  • 모바일 디바이스 스마트폰과 태블릿기기를 기울여 영상을 제어할 수 있으며, 구글 카드보드 사용하여 더욱
    몰입감 있는 영상을 감상 하실 수 있습니다.
  • VR전용 PC 헤드셋 전용 헤드셋(HMD)를 이용할 수 있습니다. 자세한 사항은 VR 전용 헤드셋 이용하기를
    참고하시기 바랍니다.

아프리카TV
VR라이브 서비스는
현재 오큘러스 DK1과
DK2를 지원합니다.

  • Windows 7/8 (파이어폭스 브라우저만 지원) 아래 단계를 참고바랍니다.
    1. 1. 다운로드 및 설치 Oculus Runtime 0.6 (Oculus Runtime v0.7의 경우 다이렉트 모드만
           지원되므로 WebVR이 지원되지 않습니다.)
    2. 2. WebVR 버전 브라우져 다운로드 및 설치 (파이어폭스 – 단독실행형 브라우저)
    3. 3. 영상이 실행된 후 플레이어의 좌측 하단 HMD모드 아이콘을 클릭한 후 헤드셋을 착용합니다.
    4. 4. (옵션) 화면이 제대로 나오지 않을 경우 "Window Key + P"를 눌러 "Projector Only" 모드로 변경
  • 헤드셋 컨트롤 VR모드의 비디오 시청중 키보드의 스페이스바 또는 게임패드의 "X" 버튼을 클릭하여
    재생/정지 할 수 있습니다. VR모드 해제시 Esc 버튼을 클릭하세요
    1. 1. 영상 재생을 시작하고 싶을 경우 스페이스바를 클릭하세요
    2. 2. 메뉴 또는 정지 아이콘을 클릭하고 싶을 경우 고개를 움직여 초점 아이콘을 위치 시킨 후
           마우스 왼쪽 버튼을 클릭합니다.
    3. 3. 화면 중앙으로 복귀하고 싶은 경우 숫자 버튼 또는 게임패드의 "B" 를 클릭하세요
